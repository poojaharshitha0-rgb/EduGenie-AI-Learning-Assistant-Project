# 🤖 EduGenie – Google Gemini Powered Learning Assistant

## 📖 Project Overview

EduGenie is an AI-powered educational learning assistant developed using *Python, **FastAPI, **HTML, **CSS, **Jinja2, and the **Google Gemini API*. The application helps students learn more effectively by providing intelligent educational assistance through multiple AI-powered modules such as Question Answering, Notes Summarization, Concept Explanation, Quiz Generation, and Personalized Learning Paths.

The primary goal of EduGenie is to make learning interactive, personalized, and accessible by leveraging Google's Gemini Large Language Model.

# ⚙️ Installation

## Step 1

Clone the repository.

bash
git clone <repository-url>


---

## Step 2

Navigate to the project folder.

bash
cd EduGenie


---

## Step 3

Install the required dependencies.

bash
pip install -r requirements.txt


---

## Step 4

Generate a Google Gemini API Key from Google AI Studio.

Replace your API key inside the project.

---

## Step 5

Run the FastAPI server.

bash
uvicorn main:app --reload


---

## Step 6

Open your browser.


http://127.0.0.1:8000
